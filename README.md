# Password Strength Checker

## Description
A Python-based password strength evaluation tool that analyzes user passwords using regex validation techniques.

## Features
- Checks minimum length requirement
- Validates uppercase and lowercase letters
- Detects numeric characters
- Ensures inclusion of special characters
- Provides improvement suggestions

## Technologies Used
- Python
- Regular Expressions (re module)

## How to Run

```bash
python password_checker.py
